# 🚀 Projeto Orion - Inteligência de Borda para CubeSats

An On-Board Computer (OBC) architecture designed for maximum energy efficiency and autonomous survival of climate-monitoring CubeSats. 

Este repositório contém a **Landing Page** oficial do projeto, desenvolvida como entrega obrigatória para a disciplina de **Front-End Design** na **Global Solution 2026**.

---
## 🛰️ Sobre o Projeto

O **Projeto Orion** propõe uma solução de computação de borda (*Edge Computing*) embarcada para nanossatélites (CubeSats). O principal diferencial do software é a **política de desperdício zero de energia**. 

A arquitetura elimina 100% dos sinalizadores visuais redundantes (LEDs físicos) do circuito, centralizando o diagnóstico em telemetria estruturada e telas de testes. O sistema gerencia de forma autónoma a rotação dos painéis solares na presença de luz e ativa o modo de segurança (*Modo Eclipse*) ao entrar na sombra da Terra, além de monitorar a saúde térmica da estrutura.
